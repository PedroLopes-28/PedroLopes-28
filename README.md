# Pedro Henrique de Souza Lopes

Desenvolvedor **Full Stack** com foco em aplicações web modernas, APIs REST e soluções orientadas a desempenho e organização de código.  
Estudante de **Ciência da Computação** na **Universidade Federal de Itajubá (UNIFEI)** e **Assessor de Projetos** na empresa júnior **byron.solutions**.

Sou um desenvolvedor com forte base em lógica, estruturas de dados e engenharia de software, atuando tanto no **front-end** quanto no **back-end**, mas tenho amor pelo **back**.  
Tenho interesse especial em arquitetura de sistemas, integração de APIs, boas práticas de desenvolvimento e aprendizado contínuo.

Atualmente, participo ativamente de projetos acadêmicos e profissionais, colaborando no planejamento, desenvolvimento e entrega de soluções tecnológicas.
